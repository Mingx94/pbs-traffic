# 警廣即時路況查詢

### 資料來源

[API Endpoint](https://od.moi.gov.tw/MOI/v1/pbs)

### 資料說明

- region(路況區域) // "A","N","M","S","E"代表"全","北","中","南","東"
- srcdetail(資料來源)
- areaNm(地區區分說明)
- UID(唯一編號)
- direction(方向)
- y1(緯度)
- happentime(發生時間)
- roadtype(路況類別)
- road(道路名稱)
- modDttm(修改時間)
- comment(路況說明)
- happendate(發生時間)
- x1(經度)

> 詳細資訊請參照 https://data.gov.tw/dataset/15221
